# RunObjectStatus

The status of the run, which can be either `queued`, `in_progress`, `requires_action`, `cancelling`, `cancelled`, `failed`, `completed`, or `expired`.


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `Queued`         | queued           |
| `InProgress`     | in_progress      |
| `RequiresAction` | requires_action  |
| `Cancelling`     | cancelling       |
| `Cancelled`      | cancelled        |
| `Failed`         | failed           |
| `Completed`      | completed        |
| `Expired`        | expired          |

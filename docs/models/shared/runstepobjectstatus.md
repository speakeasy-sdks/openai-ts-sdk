# RunStepObjectStatus

The status of the run, which can be either `in_progress`, `cancelled`, `failed`, `completed`, or `expired`.


## Values

| Name         | Value        |
| ------------ | ------------ |
| `InProgress` | in_progress  |
| `Cancelled`  | cancelled    |
| `Failed`     | failed       |
| `Completed`  | completed    |
| `Expired`    | expired      |

# Status

The current status of the fine-tuning job, which can be either `validating_files`, `queued`, `running`, `succeeded`, `failed`, or `cancelled`.


## Values

| Name              | Value             |
| ----------------- | ----------------- |
| `ValidatingFiles` | validating_files  |
| `Queued`          | queued            |
| `Running`         | running           |
| `Succeeded`       | succeeded         |
| `Failed`          | failed            |
| `Cancelled`       | cancelled         |
